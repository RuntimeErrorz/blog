# 1. 非编辑器

| 快捷键     | 效果       |
| ---------- | ---------- |
| Ctrl + J   | 隐藏底部   |
| Ctrl + B   | 隐藏侧边栏 |
| Ctrl + \   | 分屏       |
| Ctrl + 1/2 | 切换分屏   |
| Ctrl + Tab | 切换Tabs   |
| Ctrl + R   | 打开项目   |
| Ctrl + P   | 打开文件   |

# 2. 编辑器

| 快捷键           | 效果         |
| ---------------- | ------------ |
| Ctrl + Shift + P | 显示所有命令 |
| Alt + Shift + O  | 优化导入     |

# 3. 光标相关

| 快捷键              | 效果                     |
| ------------------- | ------------------------ |
| →                  | 向右移动光标             |
| Ctrl + →           | 向右移动光标到下一个词尾 |
| Shift +  →         | 向右选中                 |
| Ctrl + Shift + →   | 向右选中单词             |
| Shift + Alt + →    | 向右智能选择             |
| Ctrl + L            | 选择一行                 |
| Alt + ↑            | 向上/向下移动行          |
| Alt + Shift + ↑/↓ | 复制到上/下行            |

# 3. 多光标相关

| 快捷键             | 效果                                                                 |
| ------------------ | -------------------------------------------------------------------- |
| Alt                | 多光标                                                               |
| Ctrl + Alt + I     | 递增输入数字（需要扩展）                                             |
| Ctrl + Alt + ↑/↓ | 在上/下一行同一列添加光标                                            |
| Ctrl + D           | 添加选区到下一个查找到的匹配，如果匹配到多个，每触发一次就多添加一个 |
| Ctrl + Shift + L   | 添加选区到所有查找到的匹配                                           |
| Esc                | 取消多光标                                                           |
